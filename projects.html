Featured Projects

Spotify Controller

Key Features: LCD display with song title, artist, and playback time | Playback controls (play/pause, next, previous) | Volume control

How It Works: Combining a Python script with Arduino I was able to integrate physical inputs to control playback while receiving song details in real time.

I learned how to access the Spotify API, handle OAuth authentication, and serial communication between Python and Arduino.

Temperature Regulation Controller

Key Features: Temperature monitoring & display | Fan speed control | Automatic shutdown safeguard | Persistent mode storage

How It Works: I designed an embedded feedback system with a fan that automatically adjusts its speed/power usage based on surrounding temperature.

Failsafe: An automatic shutdown safeguard will be triggered when temperatures fall outside the selected mode's (heating/cooling) safe operating range.

Webcam Controller

Key Features: Photo capture | Adjustable zoom | Automatic display rotation

How It Works: Using a Java-based framework (Processing) paired with an Arduino, I was able to directly access my laptop's webcam and manipulate the display.
